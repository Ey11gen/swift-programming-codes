# Swift Programming Language — Source Code Repository

**A Modern Guide to iOS Development, Swift Fundamentals, App Architecture, and Real-World Programming Projects**

---

This repository contains the complete collection of code examples from every chapter of the book. Each folder corresponds to a chapter and contains all Swift playground files and source files presented in that chapter, ready to open and run in Xcode.

---

## Requirements

- macOS Sequoia (15.0) or later
- Xcode 16.0 or later (free from the Mac App Store)
- Swift 6.0 (bundled with Xcode 16)

## How to Use

1. Clone or download this repository to your Mac.
2. Open any `.swift` file in Xcode by double-clicking it, or drag it into an open Xcode Playground.
3. For chapters 14 through 20, open the `.xcodeproj` or copy the Swift files into a new SwiftUI App project as described in the book.
4. Press **Shift + Command + Return** to run a Playground, or **Command + R** to build and run a project.
